## ğŸš€ Getting Started

### Prerequisites
- Node.js installed

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/s-gowtham-reddy/meet-the-cat.git
   ```

2. **Install Dependencies**
   ```bash
   # For the server
   cd server
   npm install

   # For the client
   cd ../client
   npm install
   ```

3. **Run the Application**

   **In /server:**
   ```bash
   npm run dev
   ```

   **In /client:**
   ```bash
   npm run dev
   ```

The app will be running at `http://localhost:5173`.
